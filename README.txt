Woocoomerce plugin to display variations on the shop and category page, display them as a table on the variable product page, and more.

== Description ==

Show Product Variations For Woocommerce is Woocoomerce plugin to manage the display of variations, including the following:
<ul>
<li>Show Variations As Single Product On Shop & Category</li>
<li>Display variations as table on variable product page</li>
</ul>

== Installation ==

Go to admin in **Settings > WC SHOW VARIATION SETTINGS** just enable settings
